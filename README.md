# IonicProjectSmallBizzLedger
ionic project
#Key Features
-Real-time Dashboard: Instant calculation of current balance, total income, and total expenses.

-Offline First: Powered by Capacitor SQLite for robust local data storage on Android/iOS.

-Web Fallback: Integrated LocalStorage mechanism for seamless browser-based testing and development.

-Premium UI: A high-contrast, minimalist "Black & White" design optimized for readability in fast-paced wholesale environments.

-Automated Categorization: Quick-select categories for Sales, Marketing, Logistics, and Purchases.

#Tech Stack
-Frontend: Angular 18+ (Standalone Components, Signals API)

-Mobile Framework: Ionic Framework 8

-Database: Capacitor SQLite

-Styling: SCSS (Custom Premium Minimalism)

-Icons: Ionicons
